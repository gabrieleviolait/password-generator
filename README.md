# Password Generator

A simple, free and privacy-friendly password generator that runs directly in the browser.

## Features

- Generate random passwords
- Customize password length and character sets
- Uses the browser Web Crypto API
- Runs directly in the browser
- No account required
- No server-side storage

## Privacy

This tool runs locally in the user's browser.

Generated passwords are not intentionally sent to external services, stored on a server, or logged by this project.

Password history, if shown in the interface, is temporary for the current session and is not saved persistently by this project.
